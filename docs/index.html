  • Usually: the library to show users how to use CLI programs, which prints description, overall line, an example and a description of each individual flag. Integrates with Argufy to read auto-generated usage based on the arguments information stored in arguments.xml file. GitHub Usually1.2.0
  • Argufy: the library to parse command-line arguments to CLI programs according to a configuration object. Supports short-hand flags, boolean/numbers, commands (arguments without a flag) and multiple commands. It is also a command-line utility that allows to store arguments in arguments.xml file for automatic generation of a JavaScript file that exports extracted arguments that is compatible with Google Closure Compiler. Compatible with Documentary that prints all arguments in a table. GitHub Argufy1.4.0
